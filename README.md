# Microprocessor
Presentation of my microprocessor project using digital.
This project uses a program called Digital which can be downloaded <a href="https://github.com/hneemann/Digital">here</a>
This microprocessor is able to perform basic functions:
Arithmetic   - Addition & Subtraction
Logic        - AND & OR
Data Storing - Reading data to memory, Reading data from memory

To start the program you must enter the "microprocessor.dig" file from digital
From digital you are able to enter data into data in
From the files you are able to change memory values and view opcodes in ram_vals.txt and rom_vals.txt respectively
